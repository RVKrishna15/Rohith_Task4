# Weather App

## Description
Weather App is a beginner-friendly Python application that retrieves real-time weather information for any city using the OpenWeather API.

## Features
- Search weather by city
- Current temperature
- Humidity
- Weather condition
- Wind speed

## Technologies Used
- Python 3
- Requests
- OpenWeather API

## Installation

```bash
pip install requests
```

## Run

```bash
py weather.py
```
